# Company Logos

Place company logos in this directory.

## Logo Naming Convention

Use lowercase, hyphenated names matching the organization name:
- `euromed-innovation.png`
- `tudigisec.png`
- `sky-media.png`
- `digital-research.png`
- `freelance.png`
- `radio-gafsa.png`
- `lexi.png`

## Logo Specifications

- **Format**: PNG (preferred) or SVG
- **Size**: Recommended 512x512px or larger (square format)
- **Background**: Transparent background preferred
- **Style**: Clean, recognizable logo

## Adding New Logos

1. Add the logo file to this directory
2. Update the logo path in `src/data/portfolio.ts` for the corresponding experience entry
3. The logo will automatically display in the ExperienceApp component
